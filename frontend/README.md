# Greenfield Energy - Frontend

Modern, professional frontend for Greenfield Energy website built with React, React Router, and Tailwind CSS.

## Features

- Fully responsive design
- Modern UI/UX with smooth animations
- Multi-page application with React Router
- Professional color scheme (blue and green)
- Optimized images from Pexels
- Component-based architecture

## Installation

```bash
npm install
```

## Running the Application

### Development Mode
```bash
npm run dev
```

The application will start on http://localhost:5173

### Build for Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Pages

- **Home** - Hero section, statistics, core services overview, and CTA
- **About** - Company story, vision, mission, values, and partnerships
- **Services** - Detailed service offerings with features
- **Products** - Software product lineup (SandMaster, AutoWellz, AutoPro, RockMaster, GREEN)
- **Contact** - Contact form and company information

## Technologies Used

- React 18
- React Router DOM
- Tailwind CSS
- Vite
- Lucide React (icons)

## Design Features

- Professional blue and green gradient theme
- Smooth hover animations and transitions
- Card-based layouts with shadows
- Responsive grid systems
- Modern typography and spacing
- High-quality stock images from Pexels
